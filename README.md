# 🖼️ 썸네일 제작기 ✨

**카테고리별 감성 썸네일 자동 생성기 (Canvas 기반)**

사진 한 장과 제목만 입력하면,
카테고리별 컬러·배지·스티커가 자동으로 적용된
**블로그/인스타용 감성 썸네일**을 만들어주는 웹 도구입니다.

---

## ✨ 주요 특징

* 📸 **사진 자동 크롭 & 프레임 적용**

  * 정방형(1:1) 크롭
  * 종이 액자 느낌의 둥근 프레임
* 🏷️ **카테고리 배지 자동 생성**

  * 텍스트 길이에 따라 배지 크기 자동 조절
  * 카테고리별 포인트(dot) 색상
  * 메인 배경색(`bg`)을 기반으로 배지 배경색 자동 파생
* 🎨 **카테고리별 톤앤무드**

  * 음식 / 디저트 / 카페 / 여행 / 뷰티 / 연극 / 영화 등
  * 배경색, 배지색, 스티커가 카테고리별로 다르게 적용
* 🧁 **스티커 PNG 합성**

  * 카테고리별 스티커 자동 배치
  * 살짝 기울어진 “붙인 스티커” 느낌
* ✍️ **다양한 한글 폰트 지원**

  * 눈누·구글 폰트 기반
  * 제목 줄바꿈 지원
* 💾 **이미지 다운로드**

  * JPEG 썸네일 즉시 저장

## 🎨 배지 배경 자동 생성 로직

카테고리 배지는 **메인 배경색(`bg`)을 기준으로 자동 파생**됩니다.

* 배경색을 흰색 쪽으로 섞어 밝은 톤 생성
* 알파값을 적용해 종이 느낌 유지
* 텍스트 색도 배경에 맞춰 자동으로 어둡게 조정 가능

```js
const makeBadgeBgFromBg = (bgHex, { mix = 0.68, alpha = 0.92 } = {}) => {
  // bg → 밝은 톤 배지 색 생성
};
```

👉 덕분에 카테고리별로 색을 하나하나 맞추지 않아도
**톤온톤 디자인을 일관되게 유지**할 수 있습니다.

## 💡 제작 의도

* 블로그/인스타 썸네일을 **매번 포토샵으로 만들기 귀찮아서**
* 카테고리만 선택하면 **톤·배지·스티커가 자동으로 맞춰지도록**
* “기록”과 “일기” 느낌의 **따뜻한 썸네일 스타일**을 만들고 싶어서
